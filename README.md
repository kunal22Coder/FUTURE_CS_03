# Future Interns - Cyber Security Task 3

## API Security Risk Analysis

### Objective

Perform a read-only API Security Risk Analysis on a public API and identify potential security risks.

### API Tested

JSONPlaceholder API

https://jsonplaceholder.typicode.com

### Tools Used

* Postman
* Browser Developer Tools
* JSONPlaceholder Documentation

### Methodology

1. Reviewed API documentation
2. Tested GET endpoints
3. Inspected headers and responses
4. Assessed authentication and access control
5. Identified security risks
6. Suggested remediation measures

### Key Findings

* Open / Publicly Accessible Endpoints
* Missing Authentication Controls
* Excessive Data Exposure
* Missing Rate Limiting Indicators

### Risk Levels

* Medium
* Low
